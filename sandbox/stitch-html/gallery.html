Technical Specifications
Performance
All animations target 60fps. Code-split and lazy-loaded via dynamic import. Hero tile loads after 300ms delay to avoid LCP impact. No animation blocks critical rendering.
Accessibility
All animations respect prefers-reduced-motion: reduce. When motion is reduced, static fallback versions display. No information is conveyed by animation alone.
Registration
Each animation registers in ANIMATION_REGISTRY keyed to its route path. The AnimationTile component resolves the correct animation from the registry on mount.
